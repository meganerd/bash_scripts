# AI Agent Usage Declaration

This repository/workspace may contain content created or modified with the assistance of AI agents.

## Attribution

AI agent contributions are tracked using [Beads](https://github.com/beadnet/beads-mcp-server), 
an MCP (Model Context Protocol) server that enables transparent attribution of AI-assisted work.

## Tools Used

- **Goose** - AI coding assistant by Block
  - Model: Claude Sonnet 4.5 (via OpenRouter)
  - Extensions: Developer, Computer Controller, Fetch, Beads MCP
  
## Beads Integration

Beads MCP server is integrated to automatically track and attribute AI agent usage across:
- Code generation and modifications
- Documentation creation
- File operations
- System automation tasks

For more information about Beads and AI attribution, visit: https://github.com/beadnet/beads-mcp-server
